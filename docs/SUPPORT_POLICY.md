# Support Policy

## Current support scope

- Main branch: actively supported.
- Historical tags before `v1.0.0`: best effort only.

## Security support

- High/Critical: prioritized according to `SECURITY_CONTACTS.md`.
- Medium/Low: handled as scheduling allows.

## Reporting channels

- Usage questions: GitHub Discussions.
- Bugs: GitHub Issues.
- Vulnerabilities: GitHub Advisories or security email.
